# Rubik's-cube-Website
A rubiks's dashboard. A website prototype. 
________________________________________________________________________________________________

Made a rubki's cube dashboard. 
Interactive 3D Rubik’s Cube viewer,
Mobile-friendly responsive design.

_________________________________________________________________________________________________
# Description

The website features a fully interactive 3D Rubik’s Cube model powered by Google Model Viewer, allowing users to rotate, zoom, and interact with the cube directly in the browser.
The website includes a clean top navigation menu with sections for Home, Learn, and About, creating a smooth and user-friendly experience across desktop and mobile devices.
The interface is built using HTML, CSS, and JavaScript, with responsive design techniques and CSS custom properties (variables) for consistent styling and easier theme management. The design uses a modern dashboard-inspired layout with interactive buttons, animated 3D content, and mobile-friendly navigation.
(This website is adapts a main navigation interface)
